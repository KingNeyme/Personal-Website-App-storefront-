## CMS Build Roadmap

## Phase 1: Stabilize The Structure

- preserve the current front-facing site
- document the live content model
- define the CMS sections
- define the future folder structure

## Phase 2: Build The Admin Shell

- create staff login flow
- build dashboard layout
- build page editor navigation
- build collection management for blog posts
- build media manager shell

## Phase 3: Build The Content Layer

- create schema validation
- store singleton pages cleanly
- store posts cleanly
- add save and publish actions
- add asset references and media handling

## Phase 4: Connect The Public Site

- expose CMS-managed content to the public site
- transition one page at a time
- maintain backwards compatibility while migrating

## Phase 5: Editorial And Storefront Quality

- improve writing workflow
- improve storefront editing flow
- improve project and journey management
- improve asset handling and previews

## Phase 6: Launch Readiness

- final admin polish
- role and access checks
- content validation
- publishing workflow review
- documentation for staff use
